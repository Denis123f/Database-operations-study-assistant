Рис.1 Интерфейс программы.

Разработана для облегчения изучения языков запросов к базам данных. Имеет функцию генерации таблиц, что позволяет рандомизировать данные и улучшить процесс обучения.
Предусмотрена возможность подключения как к локальной базе данных на устройстве, так и к удалённой.

Поддерживаются следующие СУБД: PostgreSQL, SQLite. В дальнейшем планируется добавление поддержки других систем управления базами данных.
При первоначальном запуске приложение работает в режиме RAM, где по умолчанию используется SQLite, а все данные удаляются после завершения работы программы.
Поддерживается:
1) Подсветка синтаксиса;
2) Однострочные комментарии;
3) Сохранение состояния работы с последней использованной базой данных. Пользователь может прервать работу и выключить программу.
   При повторном запуске автоматически загружается последняя используемая база данных. Также реализован учебный шифратор/дешифратор.
4) Сочетание клавиш от crtl + f5-7 для быстрых активаций кнопок;
5) Система уведомлений, что оповещает пользователя о всех его действиях.
   
Работает только на linux. Сборка проекта осуществлялась с помощью CQtDeployer. После установки файла необходимо предоставить разрешение на исполнение.

Планы по развитию:
1) Сделать версию для Win;
2) Поддержка многострочных комментариев;
3) Улучшение програмнного кода шифратора и дешифратора для повышения удобства использования;
4) Визуальное отображение связей primary key, foreign key и unique key в виде стрелок при открытии проекта и переходе по кнопке в tree_database;
5) Улучшения настроек что бы увеличить их гибкость и пользовательксий опыт.
